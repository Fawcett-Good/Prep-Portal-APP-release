# Prep Portal - APP

Prep Portal is a Flutter based Android app that helps new and current students navigate campus, check events, and access school information in one centralized, user-friendly interface.

# Minimum Requirements for Installation
Android 13 (API 33) or later\
Supports for both phones and tablets (ARM64)\
iOS is **not** supported

# Installation

Prep Portal's APK download is located [here](https://github.com/Fawcett-Good/PrepPortal-APP/releases) on Github releases.\
An installation guide is also provided below to help you get started

**1. Allow Installation of Unknown Apps**\
Android may block APK installations by default.\
You **must** manually grant permissions for the specific app (e.g., browser or file manager) to install apps\
Steps may vary slightly depending on device model, but the general process is:
1) Find and open settings
2) Tap on Apps or Apps & Notifications
3) Tap on ⋮; the three-dot menu
4) Tap on Special Access or Special APP Access
5) Tap on Install Unknown APPs
6) Select the specific app you will use to install the APK (e.g., Chrome, Files, or a File Manager)
7) Enable: Allow from this source

**2. Install the APK File from Github releases**\
Download the APK file for Prep Portal [here](https://github.com/Fawcett-Good/PrepPortal-APP/releases)\
Use Chrome or a File Manager to install the APP from the APK file.
